📝 คำอธิบาย:

📝 คำอธิบาย:

javac ย่อมาจาก Java Compiler ใช้สำหรับแปลงไฟล์ .java ให้เป็น Bytecode (.class) ส่วนคำสั่ง java ใช้สำหรับรันโปรแกรม

📝 คำอธิบาย:

int (Integer) ใช้เก็บเลขจำนวนเต็ม เช่น -1, 0, 100 ส่วน double ใช้เก็บทศนิยม, String เก็บข้อความ และ boolean เก็บค่าความจริง

ใน Java, boolean มีค่าได้แค่ 2 อย่างคือ true (จริง) และ false (เท็จ) เท่านั้น (ห้ามใช้ 0, 1 หรือใส่เครื่องหมายคำพูด)

📝 คำอธิบาย:

ข้อ 22
Range ของ byte คือเท่าไหร่?
📝 คำอธิบาย:

byte มีขนาด 8-bit เก็บค่าได้ 2^8 = 256 ค่า แบบมีเครื่องหมาย (Signed) จึงเก็บได้ตั้งแต่ -128 ถึง 127

ข้อ 23
ถ้า int x = 5; int y = ++x; y มีค่าเท่าไหร่?
📝 คำอธิบาย:

++x (Pre-increment) จะเพิ่มค่า x ก่อน แล้วค่อยเอาค่าใหม่ไปกำหนดให้ y ดังนั้น x=6 และ y ก็เท่ากับ 6

ข้อ 24
ข้อใดเป็น Primitive Data Type ทั้งหมด?
📝 คำอธิบาย:

String ไม่ใช่ Primitive type (เป็น Class) ส่วน Integer, Boolean เป็น Wrapper Class. ข้อ B ถูกต้อง: int, boolean, char เป็นพื้นฐานทั้งหมด

ข้อ 25
ผลลัพธ์ของ 3 + 4 * 2 คือ?
📝 คำอธิบาย:

ต้องทำตามลำดับความสำคัญ (Order of Operations): การคูณ (*) ทำก่อนการบวก (+) ดังนั้น 4*2 = 8 แล้วค่อย +3 = 11

next() จะหยุดอ่านเมื่อเจอเว้นวรรค (Space) แต่ nextLine() จะอ่านยาวไปจนถึงการกด Enter (ขึ้นบรรทัดใหม่) ทำให้เก็บประโยคที่มีเว้นวรรคได้
