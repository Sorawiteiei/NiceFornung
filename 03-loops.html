Loops: for, while, do-while

📅 2 กุมภาพันธ์ 2026 ⏱️ อ่าน 12 นาที

🔄 Loop คืออะไร?

ลองนึกภาพว่าอาจารย์ให้เขียน "สวัสดี" 100 รอบ...
❌ แบบไม่ใช้ Loop: เขียน System.out.println("สวัสดี"); 100 บรรทัด
✅ แบบใช้ Loop: เขียนแค่ 3 บรรทัด ให้โปรแกรมวนทำซ้ำเอง!

🤔 เลือกใช้ Loop แบบไหนดี?

จำง่ายๆ แบบนี้:
• รู้จำนวนรอบแน่นอน (เช่น นับ 1-10) → ใช้ for
• ไม่รู้จำนวนรอบ แต่รู้เงื่อนไข → ใช้ while
• ต้องทำอย่างน้อย 1 ครั้งก่อน → ใช้ do-while

for loop

ใช้เมื่อ รู้จำนวนรอบที่แน่นอน:

// นับ 1 ถึง 5
for (int i = 1; i <= 5; i++) {
    System.out.println("รอบที่ " + i);
}

// Output:
// รอบที่ 1
// รอบที่ 2
// รอบที่ 3
// รอบที่ 4
// รอบที่ 5
💡 โครงสร้าง for loop: 
for (เริ่มต้น; เงื่อนไข; เพิ่ม/ลด) {
    // คำสั่งที่ทำซ้ำ
}

ตัวอย่าง: นับถอยหลัง

for (int i = 10; i >= 1; i--) {
    System.out.println(i);
}
System.out.println("ปล่อยจรวด!");

while loop

ใช้เมื่อ ไม่รู้จำนวนรอบ แต่รู้เงื่อนไขที่ต้องหยุด:

int count = 1;

while (count <= 5) {
    System.out.println("รอบที่ " + count);
    count++;
}

ตัวอย่าง: รับ input จนกว่าจะพิมพ์ "exit"

Scanner sc = new Scanner(System.in);
String input = "";

while (!input.equals("exit")) {
    System.out.print("พิมพ์อะไรก็ได้ (exit เพื่อออก): ");
    input = sc.nextLine();
    System.out.println("คุณพิมพ์: " + input);
}

do-while loop

เหมือน while แต่ ทำอย่างน้อย 1 รอบก่อน แล้วค่อยเช็คเงื่อนไข:

int num;
Scanner sc = new Scanner(System.in);

do {
    System.out.print("ใส่เลข 1-10: ");
    num = sc.nextInt();
} while (num < 1 || num > 10);

System.out.println("คุณเลือกเลข: " + num);

เปรียบเทียบ Loop ทั้ง 3 แบบ

Loop ใช้เมื่อ ทำอย่างน้อย
for รู้จำนวนรอบแน่นอน 0 รอบ (ถ้าเงื่อนไขเท็จตั้งแต่แรก)
while ไม่รู้จำนวนรอบ 0 รอบ (ถ้าเงื่อนไขเท็จตั้งแต่แรก)
do-while ต้องทำอย่างน้อย 1 รอบ 1 รอบเสมอ

break และ continue

// break - หยุดลูปทันที
for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break; // หยุดที่ 5
    }
    System.out.println(i);
}
// Output: 1 2 3 4

// continue - ข้ามรอบนี้ ไปรอบถัดไป
for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue; // ข้ามเลข 3
    }
    System.out.println(i);
}
// Output: 1 2 4 5
⚠️ ระวัง Infinite Loop!

ถ้าเงื่อนไขเป็นจริงตลอด โปรแกรมจะวนลูปไม่มีที่สิ้นสุด ต้องกด Ctrl+C เพื่อหยุด

// ❌ ตัวอย่าง Infinite Loop
while (true) {
    System.out.println("ช่วยด้วย!");
}

📝 สรุปสิ่งที่ต้องจำ

  • for - รู้จำนวนรอบ, ใช้นับเลข
  • while - ไม่รู้จำนวนรอบ, เช็คเงื่อนไขก่อนทำ
  • do-while - ทำก่อนเช็คทีหลัง
  • break - หยุดลูปทันที
  • continue - ข้ามไปรอบถัดไป
