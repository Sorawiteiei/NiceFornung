Method คืออะไร?
📝 คำอธิบาย:

Method (หรือฟังก์ชัน) คือกลุ่มของคำสั่งที่ถูกรวมไว้ด้วยกันเพื่อทำงานเฉพาะอย่าง ทำให้โค้ดเป็นระเบียบและเรียกใช้ซ้ำได้

ง่าย
ข้อ 2
คำใดใช้คืนค่าจาก Method?
📝 คำอธิบาย:

Keyword return ใช้สำหรับส่งค่ากลับจาก Method ไปยังจุดที่เรียกใช้

ง่าย
ข้อ 3
void หมายความว่า?
📝 คำอธิบาย:

void เป็น Return Type ที่ระบุว่า Method นี้จะ ไม่ส่งค่าใดๆ กลับมา (ทำงานเสร็จแล้วจบไป)

ง่าย
ข้อ 4
Parameter คือ?
📝 คำอธิบาย:

Parameter (พารามิเตอร์) คือตัวแปรที่ประกาศไว้ในวงเล็บของ Method เพื่อรอรับข้อมูลเข้ามาใช้งาน

ง่าย
ข้อ 5
static method เรียกใช้โดย?
📝 คำอธิบาย:

Method ที่เป็น static เป็นของ Class ไม่ใช่อ็อบเจกต์ จึงสามารถเรียกใช้ได้เลยผ่านชื่อ Class โดยไม่ต้อง new

ง่าย
ข้อ 6
public static void main(...) เป็น?
📝 คำอธิบาย:

main เป็น Method หลักที่เป็นจุดเริ่มต้นการทำงานของโปรแกรม Java

ง่าย
ข้อ 7
Method หนึ่งควรทำหน้าที่กี่อย่าง?
📝 คำอธิบาย:

ตามหลักการ Clean Code ที่ดี (Single Responsibility) Method ควรทำหน้าที่หลักเพียง 1 อย่าง เท่านั้น

ง่าย
ข้อ 8
การเรียก Method ใช้สัญลักษณ์อะไร?
📝 คำอธิบาย:

ใช้วงเล็บ () ต่อท้ายชื่อ Method เพื่อสั่งให้ทำงาน เช่น print()

ง่าย
ข้อ 9
Method Overloading คือ?
📝 คำอธิบาย:

Overloading คือการมี Method ชื่อเหมือนกันหลายตัวใน Class เดียวกัน แต่ต้องรับ Parameter ต่างกัน (ต่างที่จำนวนหรือประเภท)

ง่าย
ข้อ 10
Local variable อยู่ใน scope ไหน?
📝 คำอธิบาย:

ตัวแปร Local คือตัวแปรที่ประกาศใน Method และจะมองเห็นหรือใช้งานได้เฉพาะ ภายใน Method นั้นๆ เท่านั้น

ข้อ 11
int add(int a, int b){return a+b;} ผลของ add(3,5)?
📝 คำอธิบาย:

3 + 5 เท่ากับ 8

ปานกลาง
ข้อ 12
int square(int n){return n*n;} ผลของ square(4)?
📝 คำอธิบาย:

4 คูณ 4 เท่ากับ 16

ปานกลาง
ข้อ 13
void greet(){print("Hi");} มี return type?
📝 คำอธิบาย:

Return type คือคำที่อยู่หน้าชื่อ Method ในที่นี้คือ void

ปานกลาง
ข้อ 14
String getName(){return "Java";} ผลของ getName()?
📝 คำอธิบาย:

Method คืนค่า String คำว่า "Java" กลับมา

ปานกลาง
ข้อ 15
boolean isEven(int n){return n%2==0;} ผลของ isEven(4)?
📝 คำอธิบาย:

4 หาร 2 ลงตัว เศษเป็น 0 เงื่อนไข 4%2==0 เป็นจริง จึงได้ true

ปานกลาง
ข้อ 16
boolean isEven(int n){return n%2==0;} ผลของ isEven(5)?
📝 คำอธิบาย:

5 หาร 2 เหลือเศษ 1 เงื่อนไข 1==0 เป็นเท็จ จึงได้ false

ปานกลาง
ข้อ 17
int max(int a,int b){return a>b?a:b;} ผลของ max(7,3)?
📝 คำอธิบาย:

ใช้ Ternary Operator: 7 > 3 เป็นจริง จึงคืนค่า a คือ 7

ปานกลาง
ข้อ 18
Method ที่เรียกตัวเองเรียกว่า?
📝 คำอธิบาย:

Recursion (รีเคอร์ชัน) คือเทคนิคที่ฟังก์ชันเรียกใช้ตัวเองเพื่อแก้ปัญหาทำซ้ำ

ปานกลาง
ข้อ 19
int fact(int n){if(n<=1)return 1; return n*fact(n-1);} fact(3)?
📝 คำอธิบาย:

3 * fact(2) -> 3 * (2 * fact(1)) -> 3 * 2 * 1 = 6

ปานกลาง
ข้อ 20
void swap(int a,int b){int t=a;a=b;b=t;} การ swap ใน Java แบบนี้?
📝 คำอธิบาย:

Java ส่ง Primitive Type แบบ Pass by Value (ส่งสำเนาค่า) การสลับค่าภายใน Method จึงไม่มีผลกับตัวแปรภายนอก

ข้อ 21
ข้อใดเป็น Method Overloading ที่ถูกต้อง?
📝 คำอธิบาย:

ข้อ C ถูกต้องเพราะ Parameter ต่างกัน (1 ตัว vs 2 ตัว) ส่วนข้อ A, B, D ผิดเพราะ Parameter เหมือนกัน (Java แยกไม่ออก)

int sum(int...nums) เป็น?
📝 คำอธิบาย:

Varargs (Variable Arguments) ใช้สัญลักษณ์ ... เพื่อรับค่าจำนวนกี่ตัวก็ได้เข้ามาเป็น Array

ข้อ 23
int sum(int...n){...return s;} sum(1,2,3,4)?
📝 คำอธิบาย:

เป็นการหาผลรวมของ 1+2+3+4 = 10

Method signature ประกอบด้วย?
📝 คำอธิบาย:

Signature (ลายเซ็น) ของ Method ที่ใช้ระบุตัวตนคือ ชื่อ + ประเภทของพารามิเตอร์ (ไม่รวม Return type)

ข้อ 25
static int count=0; เป็น?
📝 คำอธิบาย:

ตัวแปร static จะถูกแชร์ร่วมกันทุกอ็อบเจกต์ เรียกว่า Class Variable (ถ้ามี final ด้วยถึงจะเป็น Constant)

ข้อ 26
final method คือ?
📝 คำอธิบาย:

final method ป้องกันไม่ให้คลาสลูก (Subclass) นำไปเขียนทับ (Override) เพื่อเปลี่ยนพฤติกรรม

ยาก
ข้อ 27
int fib(int n){...} fib(5)?
📝 คำอธิบาย:

ลำดับ Fibonacci: 0, 1, 1, 2, 3, 5, 8... ดังนั้นตัวที่ 5 (ถ้าเริ่ม 0, 1) คือ 5

ยาก
ข้อ 28
void modify(int[] arr){arr[0]=99;} หลังเรียก arr[0]?
📝 คำอธิบาย:

Array เป็น Object เมื่อส่งเข้า Method จะเป็นการส่ง Reference (ที่อยู่) ไป ดังนั้นการแก้ไขข้างในจึงส่งผลกระทบออกมาข้างนอกด้วย

ข้อ 29
avg(2,4,6) ได้ค่าเท่าไหร่?
📝 คำอธิบาย:

(2+4+6)/3 = 12/3 = 4.0 (มีการ cast เป็น double จึงไม่ใช่จำนวนเต็มเฉยๆ)
