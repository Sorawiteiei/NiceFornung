Control Flow: if-else และ switch

📅 2 กุมภาพันธ์ 2026 ⏱️ อ่าน 10 นาที

🤔 Control Flow คืออะไร?

ลองนึกภาพว่าเรากำลังตัดสินใจในชีวิตจริง เช่น:
- ถ้าฝนตก → เอาร่มไป
- ถ้าไม่ตก → ไม่ต้องเอา

Control Flow ก็เหมือนกัน! มันคือการสอนให้โปรแกรมตัดสินใจได้เอง ว่าถ้าเงื่อนไขเป็นแบบนี้ ให้ทำอะไร ถ้าเป็นแบบนั้น ให้ทำอะไร

🔍 ทำไมถึงสำคัญ?

ถ้าไม่มี Control Flow โปรแกรมจะทำงานเหมือนกันทุกครั้ง ไม่สามารถ:

  • ตรวจสอบว่า login ถูกหรือผิด
  • ให้เกรดตามคะแนน
  • แสดงข้อความแตกต่างกันตามเพศ/อายุ
  • ทำให้เกมมีหลายทางเลือก

🗺️ Flowchart: คิดเป็นภาพก่อนเขียนโค้ด

ก่อนจะเริ่มพิมพ์โค้ด การวาด Flowchart (ผังงาน) ช่วยให้เรามองเห็นภาพรวมการทำงานของโปรแกรมได้ชัดเจนขึ้น โดยใช้สัญลักษณ์มาตรฐานครับ:

Start / End

จุดเริ่มต้นและสิ้นสุด

Input / Output

รับค่า หรือ แสดงผล

Process

การคำนวณ / กำหนดค่า

Condition

การตัดสินใจ (True/False)

💡 ตัวอย่าง Flowchart ตรวจสอบคะแนนสอบ
Start
Score >= 50?
True (ใช่)
Print "Pass"
False (ไม่)
Print "Fail"
End

ใช้เมื่อต้องการให้โปรแกรมทำบางอย่าง เมื่อเงื่อนไขเป็นจริง

int score = 75;

if (score >= 50) {
    System.out.println("สอบผ่าน!");
}

if-else statement

เพิ่มทางเลือกเมื่อเงื่อนไข ไม่เป็นจริง

int score = 45;

if (score >= 50) {
    System.out.println("สอบผ่าน!");
} else {
    System.out.println("สอบไม่ผ่าน ต้องสอบซ่อม");
}

if-else if-else (หลายเงื่อนไข)

ใช้เมื่อมีหลายเงื่อนไขที่ต้องตรวจสอบ:

int score = 85;

if (score >= 80) {
    System.out.println("เกรด A");
} else if (score >= 70) {
    System.out.println("เกรด B");
} else if (score >= 60) {
    System.out.println("เกรด C");
} else if (score >= 50) {
    System.out.println("เกรด D");
} else {
    System.out.println("เกรด F");
}
⚠️ ข้อควรระวัง: เงื่อนไขจะถูกตรวจสอบ จากบนลงล่าง ถ้าเจอเงื่อนไขแรกที่เป็นจริง จะข้ามเงื่อนไขที่เหลือทั้งหมดเลย!

Logical Operators

ใช้รวมหลายเงื่อนไขเข้าด้วยกัน:

🎯 จำง่ายๆ:
&& (AND) = ต้องจริง ทั้งคู่ → "และ"
|| (OR) = จริง อันใดอันหนึ่ง ก็พอ → "หรือ"
! (NOT) = กลับค่า จริงเป็นเท็จ เท็จเป็นจริง → "ไม่" 
// AND (&&) - ต้องจริงทั้งคู่
if (age >= 18 && hasLicense) {
    System.out.println("ขับรถได้");
}

// OR (||) - จริงอย่างน้อยหนึ่งอัน
if (day == "Saturday" || day == "Sunday") {
    System.out.println("วันหยุด!");
}

// NOT (!) - กลับค่า
if (!isRaining) {
    System.out.println("ออกไปเที่ยวได้");
}

switch statement

ใช้แทน if-else หลายๆ อัน เมื่อต้องเปรียบเทียบ ค่าเดียวกับหลายๆ กรณี:

int day = 3;

switch (day) {
    case 1:
        System.out.println("วันจันทร์");
        break;
    case 2:
        System.out.println("วันอังคาร");
        break;
    case 3:
        System.out.println("วันพุธ");
        break;
    case 4:
        System.out.println("วันพฤหัส");
        break;
    case 5:
        System.out.println("วันศุกร์");
        break;
    default:
        System.out.println("วันหยุด");
}
💡 เคล็ดลับ: อย่าลืม break; หลังแต่ละ case! ถ้าลืมใส่ โปรแกรมจะทำ case ถัดไปด้วย (fall-through)

Ternary Operator (แบบย่อ)

เขียน if-else สั้นๆ ได้ในบรรทัดเดียว:

int score = 75;
String result = (score >= 50) ? "ผ่าน" : "ไม่ผ่าน";
System.out.println(result); // ผ่าน

📝 สรุปสิ่งที่ต้องจำ

  • if - ทำเมื่อเงื่อนไขจริง
  • if-else - มี 2 ทางเลือก
  • if-else if-else - หลายเงื่อนไข
  • switch - เปรียบเทียบค่าเดียวกับหลาย case
  • Operators: && (AND), || (OR), ! (NOT)
