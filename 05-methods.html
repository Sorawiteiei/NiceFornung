Methods (ฟังก์ชัน)

📅 2 กุมภาพันธ์ 2026 ⏱️ อ่าน 12 นาที

Method คือกลุ่มของโค้ดที่ทำงานเฉพาะอย่าง สามารถเรียกใช้ซ้ำได้หลายครั้ง ช่วยให้โค้ดอ่านง่ายและไม่ต้องเขียนซ้ำ

โครงสร้างของ Method

public static returnType methodName(parameters) {
    // คำสั่ง
    return value; // ถ้ามี return type
}

Method แบบไม่ return ค่า (void)

public class Main {
    public static void sayHello() {
        System.out.println("สวัสดี!");
    }
    
    public static void main(String[] args) {
        sayHello(); // เรียกใช้
        sayHello(); // เรียกอีกครั้ง
    }
}

Method แบบมี Parameters

public static void greet(String name) {
    System.out.println("สวัสดี " + name);
}

// เรียกใช้
greet("สมชาย");  // สวัสดี สมชาย
greet("สมหญิง"); // สวัสดี สมหญิง

หลาย Parameters

public static void printInfo(String name, int age) {
    System.out.println(name + " อายุ " + age + " ปี");
}

// เรียกใช้
printInfo("สมชาย", 20); // สมชาย อายุ 20 ปี

Method แบบ Return ค่า

public static int add(int a, int b) {
    return a + b;
}

public static double calculateArea(double radius) {
    return 3.14159 * radius * radius;
}

// เรียกใช้
int sum = add(5, 3);           // sum = 8
double area = calculateArea(5); // area = 78.54
⚠️ สำคัญ:
  • ถ้า return type เป็น void = ไม่ต้อง return
  • ถ้า return type เป็นอื่น (int, double, String) = ต้อง return ค่านั้น

ตัวอย่าง: เกรดจากคะแนน

public static String getGrade(int score) {
    if (score >= 80) return "A";
    else if (score >= 70) return "B";
    else if (score >= 60) return "C";
    else if (score >= 50) return "D";
    else return "F";
}

// เรียกใช้
System.out.println(getGrade(85)); // A
System.out.println(getGrade(45)); // F

Method Overloading

สร้าง method ชื่อเดียวกัน แต่ parameters ต่างกัน:

public static int add(int a, int b) {
    return a + b;
}

public static int add(int a, int b, int c) {
    return a + b + c;
}

public static double add(double a, double b) {
    return a + b;
}

// Java เลือก method ที่เหมาะสมเอง
add(2, 3);        // เรียก add(int, int)
add(2, 3, 4);     // เรียก add(int, int, int)
add(2.5, 3.5);    // เรียก add(double, double)
💡 ทำไมต้องใช้ Method?
  • Reusability - เขียนครั้งเดียว ใช้ได้หลายที่
  • Readability - โค้ดอ่านง่ายขึ้น
  • Maintainability - แก้ไขที่เดียวมีผลทุกที่

📝 สรุปสิ่งที่ต้องจำ

  • void = ไม่ return ค่า
  • Parameters คือตัวแปรที่รับเข้ามา
  • return ส่งค่ากลับและจบ method
  • Overloading = ชื่อเดียวกัน parameters ต่างกัน
