การเริ่มเขียนโปรแกรมด้วยภาษา Java

📅 2 กุมภาพันธ์ 2026 ⏱️ อ่าน 8 นาที 🏷️ Java, พื้นฐาน

สวัสดีเพื่อนๆ! บันทึกนี้สรุปเนื้อหาที่เราติวกันเรื่อง ภาษา Java ซึ่งเป็นภาษาที่เราใช้เรียนในวิชาการเขียนโปรแกรมพื้นฐาน อ่านแล้วงงตรงไหนทักมาถามได้เลยนะ!

ทำไมเราถึงเรียน Java?

อาจารย์เลือกให้เราเรียน Java เพราะ:

  • เข้าใจง่าย - รูปแบบการเขียนชัดเจน เหมาะกับผู้เริ่มต้น
  • Object-Oriented - เป็นพื้นฐานที่ดีสำหรับเรียนภาษาอื่นต่อ
  • ใช้งานจริงได้ - ใช้ทำ Android App และระบบต่างๆ ได้

ขั้นตอนการติดตั้ง

1. ดาวน์โหลด JDK

เข้าไปที่ Eclipse Temurin (ฟรี ไม่ต้องสมัคร) แล้วโหลด JDK เวอร์ชันล่าสุด

2. ตั้งค่า Environment Variable (Windows)

  1. กด Win + R พิมพ์ sysdm.cpl แล้ว Enter
  2. ไปที่ Advanced → Environment Variables
  3. เพิ่ม JAVA_HOME = C:\Program Files\Eclipse Adoptium\jdk-21
  4. แก้ไข Path เพิ่ม %JAVA_HOME%\bin

3. ตรวจสอบการติดตั้ง

เปิด Command Prompt แล้วพิมพ์:

java -version
javac -version

ถ้าขึ้นเลข version แสดงว่าติดตั้งสำเร็จแล้ว ✅

โปรแกรมแรกของเรา: Hello World

สร้างไฟล์ชื่อ HelloWorld.java แล้วเขียนโค้ดนี้:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("สวัสดี Java!");
    }
}
⚠️ ข้อควรระวัง:
  • ชื่อไฟล์ต้องตรงกับชื่อ class (HelloWorld.java = class HelloWorld)
  • Java แยก พิมพ์เล็ก-พิมพ์ใหญ่ (case-sensitive)
  • อย่าลืม semicolon ; ท้ายบรรทัด

วิธี Compile และ Run

// Compile (แปลงเป็น .class)
javac HelloWorld.java

// Run (รันโปรแกรม)
java HelloWorld

ตัวแปรและชนิดข้อมูล

ตัวแปรที่เราใช้บ่อยๆ ในวิชานี้:

// ตัวเลขจำนวนเต็ม
int age = 20;
int score = 85;

// ตัวเลขทศนิยม
double gpa = 3.25;
double price = 199.50;

// ข้อความ
String name = "สมชาย";
String id = "65010001";

// จริง/เท็จ
boolean isPassed = true;
boolean isAbsent = false;

การรับ Input จากผู้ใช้

ใช้ Scanner สำหรับรับข้อมูลจาก keyboard:

import java.util.Scanner;

public class InputExample {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.print("ใส่ชื่อ: ");
        String name = sc.nextLine();
        
        System.out.print("ใส่อายุ: ");
        int age = sc.nextInt();
        
        System.out.println("สวัสดี " + name + " อายุ " + age + " ปี");
        
        sc.close();
    }
}

📝 สรุปสิ่งที่ต้องจำ

  • ติดตั้ง JDK และตั้งค่า JAVA_HOME
  • ชื่อไฟล์ต้องตรงกับชื่อ class
  • javac สำหรับ compile, java สำหรับ run
  • ตัวแปรพื้นฐาน: int, double, String, boolean
  • ใช้ Scanner สำหรับรับ input
