ข้อ 1
Index ของ Array ใน Java เริ่มจากเลขอะไร?
📝 คำอธิบาย:

ใน Java (และภาษาตระกูล C) Index ของ Array จะเริ่มที่ 0 เสมอ ดังนั้นตัวแรกสุดคือ index 0

ง่าย
ข้อ 2
ข้อใดประกาศ Array ถูกต้อง?
📝 คำอธิบาย:

รูปแบบที่ถูกต้องคือ Type[] name = new Type[size]; ข้อ A ผิดที่ประเภทตัวแปร, B ขาด new, D ใช้วงเล็บผิด

ง่าย
ข้อ 3
คุณสมบัติใดใช้หาความยาวของ Array?
📝 คำอธิบาย:

Array ใช้ property .length (ไม่มีวงเล็บ) เพื่อบอกขนาด ต่างจาก String ที่ใช้ .length() และ List ที่ใช้ .size()

ง่าย
ข้อ 4
int[] arr = {1,2,3}; arr.length คือ?
📝 คำอธิบาย:

Array มีสมาชิก 3 ตัว คือ 1, 2, และ 3 ดังนั้นความยาว (length) เท่ากับ 3

ง่าย
ข้อ 5
int[] arr = {10,20,30}; arr[1] คือ?
📝 คำอธิบาย:

Index เริ่มที่ 0 ดังนั้น arr[0]=10, arr[1]=20, และ arr[2]=30

ง่าย
ข้อ 6
int[] arr = {5,10,15}; arr[0] คือ?
📝 คำอธิบาย:

arr[0] คือสมาชิกตัวแรกของ Array ซึ่งในที่นี้คือเลข 5

ง่าย
ข้อ 7
ข้อใดเป็นการกำหนดค่าให้ Array พร้อมประกาศ?
📝 คำอธิบาย:

ใน Java เราใช้ปีกกา {} ในการกำหนดค่าเริ่มต้น (Array Initializer) เช่น {1, 2, 3}

ง่าย
ข้อ 8
Array ขนาด 5 มี index สูงสุดเท่าไหร่?
📝 คำอธิบาย:

Index เริ่มที่ 0 ดังนั้นถ้าขนาด 5 ตัว Index จะเป็น 0, 1, 2, 3, 4 (สูงสุดคือ size - 1)

ง่าย
ข้อ 9
ค่าเริ่มต้นของ int[] คือ?
📝 คำอธิบาย:

เมื่อสร้าง Array ของตัวเลข (int, long, double) ค่า Default ของแต่ละช่องจะเป็น 0 (หรือ 0.0)

ง่าย
ข้อ 10
ค่าเริ่มต้นของ String[] คือ?
📝 คำอธิบาย:

String เป็น Object (Reference type) ค่า Default ของสมาชิกใน Array จึงเป็น null

ปานกลาง
ข้อ 11
int[] arr = {1,2,3}; arr[3] จะเกิด?
📝 คำอธิบาย:

Array มี index 0, 1, 2 การเรียก arr[3] (ช่องที่ 4) ถือว่าออกนอกขอบเขต จึงเกิด Error ArrayIndexOutOfBoundsException

ปานกลาง
ข้อ 12
for-each loop ใช้กับ Array แบบ?
📝 คำอธิบาย:

Syntax ใน Java ใช้เครื่องหมาย : (colon) เช่น for (Type item : array)

5 + 3 + 8 = 16

📝 คำอธิบาย:

ผลรวมคือ 20 หารด้วยจำนวนสมาชิก 4 ตัว (20/4) = 5

📝 คำอธิบาย:

ตัวเลขที่มีค่ามากที่สุดในชุดนี้คือ 9

ปานกลาง
ข้อ 16
int[] a = {9,3,7,1}; ค่าน้อยสุด?
📝 คำอธิบาย:

ตัวเลขที่มีค่าน้อยที่สุดในชุดนี้คือ 1

double[] d = new double[3]; d[0] คือ?

ค่าเริ่มต้นของตัวแปรประเภททศนิยม (double/float) คือ 0.0

ปานกลาง
ข้อ 20
boolean[] b = new boolean[2]; b[0] คือ?
📝 คำอธิบาย:

ค่าเริ่มต้นของ boolean คือ false

ยาก
ข้อ 21
2D Array: int[][] a = new int[2][3]; มีกี่ element?
📝 คำอธิบาย:

Array 2 มิติขนาด 2 แถว 3 คอลัมน์ จะมีจำนวนช่องข้อมูลทั้งหมด 2 x 3 = 6 ช่อง

ยาก
ข้อ 22
int[][] a = {{1,2},{3,4}}; a[1][0] คือ?
📝 คำอธิบาย:

แถวที่ 1 (คือแถวที่สอง {3,4}) และคอลัมน์ที่ 0 (ตัวแรก) คือเลข 3

ยาก
ข้อ 23
int[][] a = {{1,2,3},{4,5,6}}; a.length คือ?
📝 คำอธิบาย:

a.length ของ Array 2 มิติคือ "จำนวนแถว" ในที่นี้มี 2 แถว คือ {1,2,3} และ {4,5,6} ดังนั้นตอบ 2

ยาก
ข้อ 24
int[][] a = {{1,2,3},{4,5,6}}; a[0].length คือ?
📝 คำอธิบาย:

a[0].length คือ "จำนวนสมาชิกในแถวที่ 0" แถวแรกมี 1,2,3 ดังนั้นยาว 3

ยาก
ข้อ 25
int[] a = {5,2,8}; Arrays.sort(a); แล้ว a[0] คือ?
📝 คำอธิบาย:

Arrays.sort(a) เรียงลำดับจากน้อยไปมาก จะได้ {2, 5, 8} ดังนั้น a[0] คือ 2

ยาก
ข้อ 26
int[] a = {5,2,8}; Arrays.sort(a); แล้ว a[2] คือ?
📝 คำอธิบาย:

หลังจากเรียงแล้วได้ {2, 5, 8} ดังนั้น a[2] (ตัวท้ายสุด) คือ 8

ยาก
ข้อ 27
ข้อใดใช้คัดลอก Array?
📝 คำอธิบาย:

Method มาตรฐานในคลาส Arrays สำหรับคัดลอกคือ Arrays.copyOf() (ส่วน clone() ก็ใช้ได้แต่เป็น method ของ object array เอง)

ยาก
ข้อ 28
int[] a = {1,2}; int[] b = a; b[0]=9; a[0] คือ?
📝 คำอธิบาย:

Array เป็น Reference Type การกำหนด b = a คือการชี้ไปที่ Array ก้อนเดียวกัน เมื่อเปลี่ยน b ก็จะกระทบ a ด้วย ผลลัพธ์คือ 9

ยาก
ข้อ 29
int[] a = {1,2,3,4,5}; นับเลขคู่ได้กี่ตัว?
📝 คำอธิบาย:

เลขคู่ในชุดนี้มีแค่ 2 และ 4 รวมทั้งหมด 2 ตัว

ข้อ 30
char[] c = {'H','i'}; new String(c) ได้?
📝 คำอธิบาย:

การสร้าง String จาก char array จะนำตัวอักษรมาต่อกันเป็นข้อความ ได้คำว่า "Hi"
