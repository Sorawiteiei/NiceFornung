Arrays (อาร์เรย์)

📅 2 กุมภาพันธ์ 2026 ⏱️ อ่าน 10 นาที

📦 Array คืออะไร?

ลองนึกภาพว่าเราต้องเก็บคะแนนของนักเรียน 5 คน...

❌ แบบไม่ใช้ Array:
int score1=85; int score2=90; int score3=78; int score4=92; int score5=88;

✅ แบบใช้ Array:
int[] scores = {85, 90, 78, 92, 88};

เหมือนมีกล่องใบใหญ่ที่มีหลายช่อง เก็บข้อมูลได้หลายค่าในชื่อเดียว!

🎯 ใช้ Array เมื่อไหร่?

  • เก็บคะแนนนักเรียนหลายคน
  • เก็บชื่อสินค้าในตะกร้า
  • เก็บอุณหภูมิ 7 วันที่ผ่านมา
  • เก็บเลือกตั้งของผู้สมัครแต่ละคน

การสร้าง Array

// วิธีที่ 1: ประกาศแล้วกำหนดขนาด
int[] scores = new int[5]; // เก็บได้ 5 ค่า

// วิธีที่ 2: ประกาศพร้อมใส่ค่าเลย
int[] scores = {85, 90, 78, 92, 88};

// Array ของ String
String[] names = {"สมชาย", "สมหญิง", "สมศรี"};

Array ขนาด 5 จะมี index: 0, 1, 2, 3, 4

การเข้าถึงและแก้ไขค่า

int[] scores = {85, 90, 78, 92, 88};

// อ่านค่า
System.out.println(scores[0]); // 85
System.out.println(scores[2]); // 78

// แก้ไขค่า
scores[2] = 80; // เปลี่ยนจาก 78 เป็น 80

// หาความยาว
System.out.println(scores.length); // 5

วนลูปกับ Array

แบบ for loop ธรรมดา

int[] scores = {85, 90, 78, 92, 88};

for (int i = 0; i < scores.length; i++) {
    System.out.println("คะแนนที่ " + (i+1) + ": " + scores[i]);
}

แบบ for-each (ง่ายกว่า)

int[] scores = {85, 90, 78, 92, 88};

for (int score : scores) {
    System.out.println(score);
}
💡 for-each ใช้เมื่อ:
  • ต้องการอ่านค่าทุกตัวตั้งแต่ต้นจนจบ
  • ไม่ต้องการรู้ index
  • ไม่ต้องการแก้ไขค่าใน array

ตัวอย่าง: หาค่าสูงสุด

int[] scores = {85, 90, 78, 92, 88};
int max = scores[0]; // สมมติตัวแรกมากสุด

for (int score : scores) {
    if (score > max) {
        max = score;
    }
}

System.out.println("คะแนนสูงสุด: " + max); // 92

ตัวอย่าง: หาค่าเฉลี่ย

int[] scores = {85, 90, 78, 92, 88};
int sum = 0;

for (int score : scores) {
    sum += score;
}

double avg = (double) sum / scores.length;
System.out.println("คะแนนเฉลี่ย: " + avg); // 86.6

📝 สรุปสิ่งที่ต้องจำ

  • Array เก็บข้อมูลหลายค่าในตัวแปรเดียว
  • Index เริ่มจาก 0 ไม่ใช่ 1!
  • ใช้ .length หาความยาว
  • for-each ใช้อ่านค่าได้ง่าย
  • ระวัง ArrayIndexOutOfBoundsException!
