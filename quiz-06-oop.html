ข้อ 1
OOP ย่อมาจาก?
📝 คำอธิบาย:

OOP ย่อมาจาก Object Oriented Programming หรือการเขียนโปรแกรมเชิงวัตถุ ซึ่งเป็นหัวใจหลักของภาษา Java

ง่าย
ข้อ 2
Class คือ?
📝 คำอธิบาย:

Class เปรียบเสมือน พิมพ์เขียว (Blueprint) หรือแม่พิมพ์ ที่ใช้กำหนดลักษณะและพฤติกรรมเพื่อสร้าง Object

ง่าย
ข้อ 3
Object คือ?
📝 คำอธิบาย:

Object คือสิ่งที่ถูกสร้างขึ้นมาจาก Class เรียกว่า Instance มีตัวตนและสถานะจับต้องได้ใน Memory

ง่าย
ข้อ 4
คำใดใช้สร้าง Object?
📝 คำอธิบาย:

Keyword new ใช้สำหรับจัดสรรพื้นที่ในหน่วยความจำเพื่อสร้าง Object ใหม่

ง่าย
ข้อ 5
Constructor คือ?
📝 คำอธิบาย:

Constructor คือ Method พิเศษที่จะถูกเรียกทำงานอัตโนมัติเมื่อมีการ สร้าง Object (new) ใช้สำหรับกำหนดค่าเริ่มต้น

ง่าย
ข้อ 6
Encapsulation คือ?
📝 คำอธิบาย:

Encapsulation คือ การห่อหุ้มข้อมูล เพื่อซ่อนรายละเอียดภายในและป้องกันการเข้าถึงข้อมูลโดยตรงจากภายนอก (ใช้ private)

ง่าย
ข้อ 7
Inheritance คือ?
📝 คำอธิบาย:

Inheritance (การสืบทอดคุณสมบัติ) คือการที่ Class ลูกได้รับคุณสมบัติและ Method มาจาก Class แม่ ทำให้ไม่ต้องเขียนโค้ดซ้ำ

ง่าย
ข้อ 8
คำใดใช้สืบทอด Class?
📝 คำอธิบาย:

ใช้คำสั่ง extends เพื่อระบุว่า Class นี้สืบทอดมาจาก Class อื่น เช่น class Dog extends Animal

ง่าย
ข้อ 9
this ใช้อ้างถึง?
📝 คำอธิบาย:

this เป็น Reference ที่ชี้ไปยัง Object ตัวปัจจุบัน ที่กำลังทำงานอยู่

ง่าย
ข้อ 10
super ใช้อ้างถึง?
📝 คำอธิบาย:

super ใช้สำหรับอ้างถึงสมาชิก (ตัวแปรหรือ Method) ของ Parent Class (คลาสแม่)

ปานกลาง
ข้อ 11
private เข้าถึงได้จาก?
📝 คำอธิบาย:

private เป็นระดับการเข้าถึงที่จำกัดที่สุด เข้าถึงได้เฉพาะ ภายใน Class ตัวเอง เท่านั้น

ปานกลาง
ข้อ 12
public เข้าถึงได้จาก?
📝 คำอธิบาย:

public เปิดกว้างที่สุด สามารถเข้าถึงได้จาก ทุกที่ ในโปรเจกต์

ปานกลาง
ข้อ 13
protected เข้าถึงได้จาก?
📝 คำอธิบาย:

protected ยอมให้ Package เดียวกันเข้าถึงได้ และยอมให้ Subclass ต่าง Package เข้าถึงได้ด้วย

ปานกลาง
ข้อ 14
Getter/Setter ใช้ทำอะไร?
📝 คำอธิบาย:

เป็น Method สาธารณะ (public) ที่ใช้สำหรับ อ่าน (get) หรือ แก้ไข (set) ค่าของตัวแปร private อย่างปลอดภัย

ปานกลาง
ข้อ 15
Method Overriding คือ?
📝 คำอธิบาย:

Overriding คือการที่ Class ลูกเขียน Method ซ้ำกับ Class แม่ เพื่อเปลี่ยนแปลงการทำงานให้เป็นแบบของตัวเอง

ปานกลาง
ข้อ 16
Polymorphism คือ?
📝 คำอธิบาย:

Polymorphism (พอลินอร์ฟิซึม) คือความสามารถที่ Object หนึ่งๆ สามารถแสดงออกได้ หลายรูปแบบ หรือหลายพฤติกรรม

ปานกลาง
ข้อ 17
abstract class คือ?
📝 คำอธิบาย:

Abstract Class คือคลาสที่เป็นเพียงโครงร่าง ไม่สมบูรณ์ จึง ไม่สามารถนำไปสร้าง Object (new) โดยตรงได้ ต้องสืบทอดไปใช้

ปานกลาง
ข้อ 18
interface คือ?
📝 คำอธิบาย:

Interface เปรียบเสมือน สัญญา (Contract) ที่กำหนดว่า Class ที่นำไปใช้จะต้องมี Method อะไรบ้าง แต่ไม่ได้กำหนดวิธีการทำงานภายใน

ปานกลาง
ข้อ 19
คำใดใช้กับ interface?
📝 คำอธิบาย:

Class ใช้คำสั่ง implements เพื่อนำ Interface เข้ามาใช้งาน (Implement)

ปานกลาง
ข้อ 20
Java รองรับ multiple inheritance ผ่าน?
📝 คำอธิบาย:

Java ไม่ยอมให้ extends หลาย Class แต่ยอมให้ Class หนึ่ง implements ได้หลาย Interface เพื่อแก้ปัญหา Multiple Inheritance

ข้อ 21
Animal a = new Dog(); เรียกว่า?
📝 คำอธิบาย:

Upcasting คือการมอง Object ลูก (Dog) ในฐานะ Object แม่ (Animal) เป็นการแปลงขึ้นบน

ยาก
ข้อ 22
instanceof ใช้ทำอะไร?
📝 คำอธิบาย:

Operator instanceof ใช้ตรวจสอบว่า Object นั้นถูกสร้างมาจาก Class หรือ Interface ที่ระบุหรือไม่

ยาก
ข้อ 23
หลักการ OOP ที่สำคัญมี 4 ข้อคือ?
📝 คำอธิบาย:

4 เสาหลักของ OOP คือ: 1.Encapsulation 2.Inheritance 3.Polymorphism 4.Abstraction

ยาก
ข้อ 24
Constructor ต้องมีชื่อเหมือน?
📝 คำอธิบาย:

กฎเหล็กของ Constructor คือต้องมีชื่อ ตรงกับชื่อ Class เป๊ะๆ (และไม่มี return type)

ยาก
ข้อ 25
final class คือ?
📝 คำอธิบาย:

คลาสที่ประกาศเป็น final จะเป็นหมัน คือ ห้ามใครมาสืบทอด (extends) ต่อไปอีก

ยาก
ข้อ 26
@Override annotation ใช้ทำอะไร?
📝 คำอธิบาย:

ใส่ไว้เพื่อบอก Compiler (และโปรแกรมเมอร์) ว่า Method นี้ตั้งใจจะทับของพ่อนะ ถ้าชื่อผิด Compiler จะได้แจ้งเตือน

ยาก
ข้อ 27
toString() method มาจาก class ใด?
📝 คำอธิบาย:

ทุก Class ใน Java สืบทอดมาจาก java.lang.Object ซึ่งมี Method มาตรฐานอย่าง toString() ติดตัวมาด้วยเสมอ

ยาก
ข้อ 28
static variable ใน class คือ?
📝 คำอธิบาย:

ตัวแปร static จะมีเพียงสำเนาเดียวใน Memory และถูกแชร์ให้ทุกๆ Object ของ Class นั้นใช้ร่วมกัน

ยาก
ข้อ 29
Default constructor คือ?
📝 คำอธิบาย:

ถ้าเราไม่ได้ประกาศ Constructor เลย Java จะแถม "Default Constructor" (แบบไม่มี Parameter) ให้ฟรีๆ

ยาก
ข้อ 30
ทุก class ใน Java สืบทอดจาก?
📝 คำอธิบาย:

java.lang.Object คือบรรพบุรุษสูงสุดของทุก Class ใน Java (ถ้าไม่ extends อะไร มันจะ extends Object นัยๆ)
